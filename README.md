# InfernalToast Web 
 
 

## Deployed on Vercel
https://infernal-simple-dapp.vercel.app/


This dapp works on Kovan or Mainnet and automatically detects chainId from metamask 


### Development commands
```
nvm install 14
nvm use 14
npm install
npm run dev
```

### Packaging commands
```
npm run build
npm run server
```
